PHP
    Необходимо разработать скрипт который будет синхронизировать, сохранять данные товаров из файла csv в базу.

    Этот скрипт будет запускаться через крон каждые 2 минуты.

    Обновлять или создавать товары по id. 
    Товаров в файле может быть до 800 000.

    Полей для обновления может быть до 50, но в рамках тестового пусть будет 3, id, title, price

    Можно использовать любой фреймворк и версию PHP.
    Для базы данных MySql

    Результат можно скинуть в зипе или через гит.
